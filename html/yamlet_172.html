resolve.classified R Documentation

Resolve Classified

Description

Resolves classified. A non-operation since classified already has an explicit guide. Useful to make resolve() idempotent for 'dvec'.

Usage

## S3 method for class 'classified'
resolve(x, ...)

Arguments

x

classified
...

ignored

Value

classified
