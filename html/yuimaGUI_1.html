yuimaGUI R Documentation

User-friendly Interface for the yuima package

Description

Runs yuima Graphical User Interface

Usage

yuimaGUI(theme = "black")

Arguments

theme

GUI theme: "black" or "white".

Value

Starts yuima GUI

Author(s)

The YUIMA Project Team

Examples

## Not run: 
yuimaGUI()

## End(Not run)
