ds.skew R Documentation

Calculating skewness for numeric data

Description

Calculating Pearson's skewness in three types: mode, median, and mean.

Usage

ds.skew(x, type = 3)

Arguments

x

:a numeric variable
type

:type = 1 for mode skewness; type = 2 for median skewness; type = 3 for mean skewness

Examples

ds.skew(mtcars[,1])
