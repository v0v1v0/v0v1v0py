unclassified.dvec R Documentation

Unclassify Decorated Vector

Description

Unclassifies dvec. A non-operation, since dvec is not classified. Needed for completness by resolve.dvec.

Usage

## S3 method for class 'dvec'
unclassified(x, ...)

Arguments

x

dvec
...

ignored

Value

dvec
