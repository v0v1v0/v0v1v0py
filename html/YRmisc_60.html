pt.semivar R Documentation

Semivariance of loss

Description

This function give the semivariance of a losing scenario.

Usage

pt.semivar(r,p)

Arguments

r

:a vector of periodic returns
p

:target return

Examples

rt <- runif(12,-1,1) # generate random number to simulate returns
pt.semivar(rt,0)
pt.semivar(rt,0.03)
